# Welcome to DEPL

DEPL is a powerful deep linking platform that helps you create, manage, and track mobile app deep links.

## Features

- **Universal Links**: Support for iOS Universal Links
- **App Links**: Support for Android App Links
- **Analytics**: Real-time click tracking and analytics
- **Custom Domains**: Use your own subdomain
- **REST API**: Full-featured API for integration

## Quick Start

Get started with DEPL in minutes:

1. Sign up for a free account
2. Create your workspace
3. Connect your iOS/Android apps
4. Start creating deep links
